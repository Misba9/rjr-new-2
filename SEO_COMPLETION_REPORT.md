### How to Add a New Location Page
1. Create `src/pages/locations/[LocationName]Page.tsx`
2. Copy structure from existing location page
3. Update: H1, meta tags, schema (areaServed), content
4. Import in `src/App.tsx`
5. Add route case in `renderPage()` switch
6. Add link in `src/components/Footer.tsx`
7. Add URL to `public/sitemap.xml`
8. Test locally: `npm run dev`
9. Build & deploy

### How to Add a New Blog Post
1. Create component in `src/pages/blog/`
2. Add routing in App.tsx (or use dynamic routing)
3. Update sitemap.xml
4. Add schema: Article, BlogPosting
5. Add meta tags (title, description, OG)
6. Internal link from BlogPage.tsx
